## Setup

Make sure to install the dependencies:

# npm
npm install

Also don't forget to set the environnment variables: 

OPENAI_API_KEY=
OPENAI_ASSISTANT_ID=


## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```
